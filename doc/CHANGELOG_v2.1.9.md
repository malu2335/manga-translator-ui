# v2.1.9 更新日志

发布日期：2026-03-18

## ✨ 新功能

- 新增翻译设置项 `keep_lang`（保留源语言）：在文本框合并后按源语言过滤后续处理区域；适合英文版日漫只处理英文文本，减少日文标题、拟声词等被误翻译或误擦除的情况。

## 🐛 修复

- 修复了泰语渲染异常，添加了对泰语渲染的支持。
- 修复部分带 `ICC` 色彩配置的 JPG 图片在翻译、修复或编辑器导出后出现偏色的问题：现在会在统一保存链路中尽量保留原图的色彩配置与 `DPI` 信息，减少导出结果与原图颜色不一致的情况。
- 修复“竖排内横排”关闭后仍会在竖排无断句布局中自动添加横排标记的问题：现在关闭该选项时，不会再额外写入 `<H>` 标记。
- 修复 Qt UI 中“导入翻译并渲染”预处理阶段忽略字符串图片路径的问题：`load_text` 预导入 TXT 到 JSON 时现在同时支持 `str` / `os.PathLike` 输入，不再因为输入不是带 `.name` 的图片对象而跳过 `manga_translator_work/originals/` 或 `translations/` 下的 TXT 文件。

## 📝 文档与说明

- 新增“保留源语言”文档说明：补充 Qt / Web 配置项 `keep_lang` 的用途、推荐用法和中日共用汉字的保留规则。
