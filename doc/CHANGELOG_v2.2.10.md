# v2.2.10 更新日志

发布日期：2026-06-26

## ✨ 新功能与优化

### 快捷键支持
- 支持在编辑器中通过 `Ctrl+A` 快速全选当前图片的所有文本框。

### 界面视觉统一
- 统一了二级页面（如设置、API管理、提示词与替换规则编辑器等）的主题风格，解决了深浅色切换时的部分背景残留问题。
- 优化了主要按钮的颜色表现，将“开始翻译”等核心操作按钮恢复为更易识别的蓝色设计。
- 优化了侧边栏和设置页的布局，减少了界面视觉噪声。

### API 配置面板优化
- 多通道配置升级为卡片式分组，排版更加整洁。
- 当所选的模型或接口不需要 API Key 时，会直观显示空状态提示，区分层次。

## 🐛 修复与改进

### OCR 与渲染修复
- 修复了 PaddleOCR-VL 偶尔产生重复符号或字符串幻觉的问题。
- 修复了启用气泡英文排版时，可能因写入只读属性导致任务中断的问题。
- 修复了 AI 渲染默认提示词只清除编号框未清除数字的问题。
- 为 `balloon_fill` 排版模式接入 AI 断句自动扩大文字逻辑，可复用已有 `[BR]` 组合优化以减少不必要断行、提升可用字号。

### 图像兼容性提升
- 修复了输入带透明通道或非标准格式（如 RGBA）的图片在进行 AI 上色、渲染或超分辨率处理（Real-CUGAN等）时可能崩溃或异常的问题。

### 编辑器与配置体验修复
- 修复了属性面板中“原文”和“译文”多行文本框高度被全局 CSS 压缩为单行的问题。
- 修复了在 API 面板中快速连续输入导致部分修改丢失的问题。
- 修复了修改 API 配置后，切换回设置页或编辑器时选项未实时同步更新的问题。
- 修复了字体管理预览在不同字重（Medium/Bold/Heavy）之间切换无效且卡顿的问题。
- 修复了 Sakura 翻译器配置项显示不全的 Bug。

## 🛠️ 底层变更
- 重新整理了 Qt UI 的内部代码结构与样式管理，以提升后续的维护与打包效率。
